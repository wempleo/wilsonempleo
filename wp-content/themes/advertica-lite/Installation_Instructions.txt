/*========= Installation =========*/

1. Upload the `advertica-lite` folder to the `/wp-content/themes/` directory
Activation and Use
1. Activate the Theme through the 'Themes' menu in WordPress
2. See Appearance -> Advertica Options to change theme options.
3. create a page and set Home Page Template for front page.
3. See Settings -> Reading -> Choose a Page for the Front page and Page for Posts page from drop-down.


/*========= Theme Features and Documentation =========*/
